## Responsive Behavior

```
Mobile (< 768px):
- Single column layout
- Bottom navigation
- Hamburger menu
- Stacked cards

Tablet (768px - 1024px):
- Two column layout
- Bottom navigation
- Hamburger menu
- Side-by-side cards

Desktop (> 1024px):
- Multi-column layout
- Top navigation only
- All features visible
- Grid layouts
- Footer visible
```
